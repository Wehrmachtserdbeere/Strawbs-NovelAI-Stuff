####

How to import Lorebooks

1. Click the book icon above your input box
2. At the top-left, next to the "Lorebook" text, you see
	an arrow pointing up into a box. Click that button.
3. Choose the .lorebook file you have downloaded.


####

Intended use

The "General Anime Lorebook" is supposed to be used with
	the Euterpe model and no module. However, it should
	not be a problem to use toehr models or modules. But
	keep in mind that it might respond differently.
